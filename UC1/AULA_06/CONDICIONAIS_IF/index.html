Estrutura Condicional IF

A estrutura if permite executar um bloco de código somente quando uma condição for verdadeira. 

let idade = 18;

if (idade >= 18) {
    console.log("Maior de idade");
}

Como funciona?

  • O JavaScript avalia a condição.
  • Se for true, executa o bloco.
  • Se for false, ignora o bloco.

Exemplo com IF e ELSE

let nota = 6;

if (nota >= 7) {
    console.log("Aprovado");
} else {
    console.log("Reprovado");
}
💡 O if é uma das estruturas mais utilizadas para tomada de decisões em aplicações JavaScript.
