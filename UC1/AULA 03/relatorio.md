# Sistema d Informação

um sistema é um conjunto de partes que trabalham juntas para atingir um objetivo.
<!-- Entrada , processo e saida -->

Entrada: 
Dados ou recursos que entram no sistema

saida:

Sistema que coleta, armazena, processa e distribui informalçoes para apoiar decisões e controle em uma organização. Ex: E-commerce, banco digital, sistema hospitalar.

Dado = fato bruto, sem contexto ou significado. Por si só, o dado n diz nada
