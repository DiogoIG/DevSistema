# SISTEMA DE LANHOUSE


## Problema
Controle sobre os sites acessados pelas pessoas, proibição na danificação das maquinas e limitação nos acessos de certos sites.

## Objetivo
Permitir acesso a internete e a jogos que o cliente deseja, podendo fazer trabalhos escolares ou fazer algumas pesquisas.

## Usuários
-Responsavel local (ADM)
-Logins de visitante para os clientes
-Clientes podendo acessar os proprios emails (opcional)

## Regras de Negócio
-Limite de tempo por determinado valor
-Não permitir uso de bebidas alcolicas
-Não permitir uso de cigarros
-Silencio para não atrapalhar a pessoa ao lado
-Uso conciente da maquina


## Requisitos Funcionais
-Sistema deve contabilizar o numero de horas q certo cliente esta a utilizar a maquina
-Sistema deve notificar o ADM caso o cliente cometa um erro no computador ou acesse sites indevidos
-Sistema d gerar liberação no computador por pagamento 
-Sistema deve gerar uma opção de monitoramento a distancia para o dono (acessar o pc d um certo cliente pelo pc do ADM)


## Requisitos Não Funcionais

-Sistema tem q ter interface não muito complexa
-Sistema tem q ser funcional em multiplos computadores
-Sistema tem q possuir um catalogo que mostre os computadores que estao sendo utilizados
-Sistema temq  mostrar horas restantes d cada pessoa no computador
